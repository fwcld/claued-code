## 구현 완료 기능

### 포커스 세션 (`focus.html` + `app.js`)
- 카테고리별 타이머: 공부📚 / 업무💼 / 휴식☕ / 운동🏃 / 기타✨
- 시작 / 일시정지 / 정지(기록) 흐름
- 세션 저장 시 메모 입력 가능
- 오늘 세션 목록 (시간·카테고리·소요시간·메모, 개별 삭제)
- Chart.js 도넛 차트 (카테고리별 비율, 30초 이상 세션부터 표시)
- **1년 집중 품질 히트맵** (GitHub 잔디 스타일, 요일별 세로 배치)
- **컨텍스트 스위칭 배지** — 이전 세션과 카테고리가 다를 때 "📚→💼" 형식 표시
- 뽀모도로 알림 (25분마다 Web Notifications API)
- 오늘의 목표 CRUD (체크박스 완료, 삭제)
- localStorage 키 네임스페이스: `fs_*`

### 마켓 세션 (`stock.html` + `stock.js`)
- 포트폴리오 요약 (총 투자금액, 평가금액, 평가손익, 실현손익)
- 투자 종목: 매수가·수량·현재가 입력 → 수익률 자동계산, 인라인 현재가 수정
- 관심 종목: 종목명 + 메모
- 판매 기록: 매수가·매도가·수량 → 실현손익·수익률 자동계산
- 유효성 검사 실패 시 빨간 테두리(`.input-error`) 시각 피드백
- localStorage 키 네임스페이스: `st_*`

### 바디 세션 (`weight.html` + `weight.js`)
- 목표 칼로리 설정 (변경 시 즉시 반영)
- 식사 기록: 시간대(아침/점심/저녁/간식) 버튼 + 음식명 + 칼로리
- 식사 목록 시간대별 그룹핑 + 각 그룹 소계
- 운동 기록: 종류 select(런닝/걷기/사이클/수영/웨이트/기타) + 거리 + 칼로리 자동계산
  - `KCAL_PER_KM = { 런닝:60, 걷기:40, 사이클:35, 수영:70, 웨이트:0, 기타:0 }`
  - 웨이트·기타는 거리 필드 자동 숨김, 칼로리 직접 입력
- 오늘 요약: 섭취/소모/순 칼로리, 총 런닝거리 + 진행 바 (80%=주황, 100%=빨강)
- 오늘 타임라인: 식사+운동 시간순 정렬, 섭취(+)/소모(-) 색상 구분
- localStorage 키 네임스페이스: `bd_*`

## CLAUDE.md 주요 규칙 (반드시 숙지)

1. **언어**: 모든 응답·설명·코드 주석은 한국어
2. **커밋**: 파일 수정 후 즉시 커밋·푸시 (요청 없어도 자동)
3. **구현 전 인터뷰 필수**:
   - 총 5개 질문, 시작 시 "총 5개 질문입니다" 안내
   - 각 질문 = 객관식 4개 보기 + 주관식 1개
   - 추천 선택지 있으면 보기 끝에 **(추천)** 표시
   - 인터뷰 후 plan spec 파일 작성 → 사용자 확인 → 구현
   - 사소한 수정(오타, 한 줄 변경)은 인터뷰 생략 가능
4. **Git PATH**: `$env:PATH = $env:PATH + ";C:\Program Files\Git\bin;C:\Program Files\Git\cmd"` 를 모든 git 명령 앞에 붙여야 함

## 시도했으나 실패하거나 보류된 것

| 항목 | 상태 | 이유 |
|------|------|------|
| **멀티유저 (닉네임 로그인)** | 보류 | Firebase 연동 필요 — 사용자가 "나중에 하겠다"고 연기 |
| **차트 미표시** | 사용자 혼동 (버그 아님) | 세션 저장 + 30초 이상 조건 필요, 코드는 정상 |

## 과거에 발생한 버그 및 해결법

| 버그 | 원인 | 해결 |
|------|------|------|
| 주식 추가 버튼 미작동 | 유효성 실패 시 피드백 없이 `return` | `markInvalid()` + `.input-error` 빨간 테두리 추가 |
| 입력 UI 레이아웃 깨짐 | 5열 그리드에 버튼이 튀어나옴 | `stock-form` flex column + `form-submit-btn` align-self 구조로 교체 |
| git 명령 실패 | Git이 기본 PATH에 없음 | PowerShell에서 PATH 수동 선행 등록 |
